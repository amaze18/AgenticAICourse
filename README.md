## üéØ Why Agentic AI?

Traditional AI education often stops at models and theory.  
**Agentic AI** goes further:

- LLMs that **plan, reason, and act**
- AI systems that **use tools, APIs, memory, and external knowledge**
- Architectures used in **real production systems** (not demos)
- Skills directly applicable to **jobs, startups, and research**

This course treats **LLMs as a computational core**, not just text generators.

> **Mental model**  
> - LLM = CPU  
> - Prompt + context = program  
> - Tools = I/O devices  
> - Memory = working & long-term state  
> - Agent loop = operating system for intelligence
